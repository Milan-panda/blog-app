
# Blog App

Full Stack Application on MERN.



## Run Locally

Clone the project

```bash
  git clone https://github.com/Milan-panda/blog-app.git
```

Go to the project directory

```bash
  cd frontend
  cd backend
```

Install Dependencies

```bash
  npm install
```

Start the server in both folder using

```bash
  npm start
```
